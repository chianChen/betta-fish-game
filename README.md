# 🐟 鬥魚養成遊戲

一個結合遺傳學、養成系統和競賽機制的瀏覽器遊戲。

## ✨ 遊戲特色

### 🧬 遺傳系統
- **顏色基因**: 紅、藍、白、黑、綠、紫、金（稀有度不同）
- **鰭型基因**: 半月、面紗、冠尾、短尾、三角
- **體型基因**: 小、中、大
- **基因遺傳**: 配對時隨機從父母各取一個等位基因

### 🌱 成長系統
- **魚苗階段** (0-300秒): 體型50%，收入 $5/分
- **幼魚階段** (300-600秒): 體型80%，收入 $10/分
- **成魚階段** (600秒+): 體型120%，收入 $20/分

### 🍽️ 飼養系統
- **飢餓度**: 隨時間增加，超過70%會降低健康
- **健康度**: 影響比賽分數和魚的價值
- **餵食**: 消耗飼料降低飢餓度，增加健康
- **水質管理**: 水質下降影響魚的健康

### 🏆 競賽系統
- **銅牌賽**: $100 報名費，獎金 $200/$100/$50
- **銀牌賽**: $300 報名費，獎金 $600/$300/$150
- **金牌賽**: $500 報名費，獎金 $1200/$600/$300
- **評分機制**: 考慮基因稀有度、成長階段、健康狀況

### 💰 經濟系統
- 每條魚根據成長階段產生被動收入
- 出售魚獲得金錢（價值基於基因、成長、冠軍數）
- 購買新魚、飼料、換水都需要金錢
🎮 遊戲操作指南
基本操作
點擊魚: 查看詳細資訊
餵食: 在資訊面板點擊餵食按鈕，或使用"餵食全部"
配對: 啟用配對模式，選擇兩條成年魚配對（需 $200）
比賽: 選擇魚參加比賽，獲勝可得獎金和冠軍頭銜
出售: 出售不需要的魚換取金錢
進階策略
選擇性繁殖: 配對高稀有度的魚培育更有價值的後代
健康管理: 保持魚的健康和飽食度以提高比賽表現
成長投資: 等待魚成長到成魚階段再出售或比賽
經濟平衡: 在購買新魚、飼料和參賽之間找到平衡
遊戲提示
魚苗無法配對和參賽
飢餓度高會降低健康和比賽分數
水質低會影響所有魚的健康
成魚的收入和價值遠高於魚苗
冠軍魚的價值會顯著提升
